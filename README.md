# mark
mark是一个mac下的免费开源的markdown编辑程序，使用[tui.editor](https://nhnent.github.io/tui.editor/)和electron构建。

# 主要特色

+ 计划todo管理
+ 支持天，周，月，季，年级别的todo定制
+ note，笔记管理
+ category， 支持分类 
+ save data on local disk，本地数据存储，支持导入导出数据文件
